# file.isFile

The `isFile` function is used to determine if a given path represents a file.

## Sample

```javascript
var tf = file.isFile("myFile.txt");
```

## API

| Calling | Returning |
|---|---|
| `file. isFile ( path )` | `Boolean` |

| Parameters | Type | Description |
|---|---|---|
| `path` | `String` | The relative path to the storage. |
