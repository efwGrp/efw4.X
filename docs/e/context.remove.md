# context.remove

The `remove` function stores data in the context.

## Sample

```javascript
context.remove("BROADCAST_CONTENT");
```
## API

| Calling | Returning |
|---|---|
| `context. remove ( key )` | `void` |

| Parameters | Type | Description |
|---|---|---|
| `key` | `String` | The key used to identify the context information. |
