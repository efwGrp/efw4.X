# グローバルイベント

```javascript
////////////////////////////////////////
// web/WEB-INF/efw/event/global.js
////////////////////////////////////////
var global = {};
global.fire = function() {
  // ...
};
```

### イベント変数

イベント変数は、ファイル名と同じ "global" という名前にする必要があります。

### fireメソッド

グローバルイベント内では、セッションオブジェクトを操作しないように注意してください。

### イベントの戻り値

グローバルイベントは何も返しません。
