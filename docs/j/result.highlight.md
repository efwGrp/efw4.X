# Result.highlight

`highlight` 関数は、クライアント側で要素をハイライト表示します。

## サンプル

```javascript
var result = new Result();
result.highlight("#text1");
```

## API

| 呼び出し | 戻り値 |
|---|---|
| `result. highlight ( selector )` | `Result` |

| パラメータ | 型 | 説明 |
|---|---|---|
| `selector` | `String` | ハイライト表示する要素を識別するjQueryセレクタ。 |
