# Excel.removeSheet

`removeSheet`関数は、シートを削除するために使用されます。

## サンプル

```javascript
var excel = new Excel("test.xlsx");
excel.removeSheet("Sheet1");
```

## API

| 呼び出し | 戻り値 |
|---|---|
| `excel. removeSheet ( sheetName )` | `Excel` |

| パラメータ | 型 | 説明 |
|---|---|---|
| `sheetName` | `String` | シート名。 |
