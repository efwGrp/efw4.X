# EFW框架

一个面向 Web 的企业级框架，采用目标导向的方法，使用 Ajax 和服务器端 JavaScript 构建快速的 Web 系统。

- **提高开发效率** - 减少重复编码工作，显著缩短开发周期，快速响应需求变更

### [客户端功能分析](client.magic.md)
EFW 框架的主要目标是提供**客户端（浏览器）和服务器之间的事件驱动通信**，以及随之而来的**用户界面操作的自动化**。

### [服务器端功能分析](server.magic.md)
Efw 框架的服务器端是一个**事件驱动型中间件**，它接收从客户端 `Efw("eventId")` 调用开始的 AJAX/WebSocket 请求。它通过 Java 和服务器端 JavaScript 的强大协作，集成了身份验证、验证、逻辑执行和资源管理。
