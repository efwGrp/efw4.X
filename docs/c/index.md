# EFW框架

## 革命性的Web开发解决方案

采用目标导向方法，通过Ajax和服务器端JavaScript高速构建Web系统


## 介绍

- **革命性的开发方法** - EFW采用目标导向方法，通过Ajax和服务器端JavaScript构建高速Web系统
- **大幅减少代码量** - 引入EFW可显著降低Web开发的复杂度，其特性包括大幅减少代码量、快速响应需求评审与规格变更等
- **提高开发效率** - 减少重复编码工作，显著缩短开发周期，快速响应需求变更
