# Excel.hideSheet

The `hideSheet` function is used to hide a sheet.

## Sample

```javascript
var excel = new Excel("test.xlsx");
excel.hideSheet("mySheet");
```

## API

| Calling | Returning |
|---|---|
| `excel. hideSheet ( sheetName )` | `Excel` |

| Parameters | Type | Description |
|---|---|---|
| `sheetName` | `String` | The sheet name. |
