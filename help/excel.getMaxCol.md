<H1>Excel.getMaxCol</H1>

The getMaxCol function is established to get the last col number starting from 1.

<h2>Sample</h2>
<pre>
	var excel = new Excel("test.xlsx");
	var maxcol = excel.getMaxCol("Sheet1");
</pre>

<tr><th>Calling</th><th>Returning</th></tr>

</table>
