# request.get

The `get` function retrieves data from the request.

## Sample

```javascript
var defId = request.get("defId");
```

## API

| Calling | Returning |
|---|---|
| `request. get ( key )` | `{String}` |

| Parameters | Type | Description |
|---|---|---|
| `key` | `String` | Identifies the requested information. |
