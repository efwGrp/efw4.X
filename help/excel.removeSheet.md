# Excel.removeSheet

The `removeSheet` function is used to remove a sheet.

## Sample

```javascript
var excel = new Excel("test.xlsx");
excel.removeSheet("Sheet1");
```

## API

| Calling | Returning |
|---|---|
| `Excel.removeSheet(sheetName)` | `Excel` |

| Parameters | Type | Description |
|---|---|---|
| `sheetName` | `String` | The sheet name. |
