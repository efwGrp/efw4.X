# Result.show

The `show` function displays elements on the client-side.

## Sample

```javascript
var result = new Result();
result.show("#table1");
```

## API

| Calling | Returning |
|---|---|
| `result. show ( selector )` | `Result` |

| Parameters | Type | Description |
|---|---|---|
| `selector` | `String` | A jQuery selector identifying the elements to be shown. |
