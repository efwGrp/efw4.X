# Excel.showSheet

The `showSheet` function is used to show a sheet.

## Sample

```javascript
var excel = new Excel("test.xlsx");
excel.showSheet("mySheet");
```

## API

| Calling | Returning |
|---|---|
| `Excel.showSheet(sheetName)` | `Excel` |

| Parameters | Type | Description |
|---|---|---|
| `sheetName` | `String` | The sheet name. |
