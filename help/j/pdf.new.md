# new Pdf

コンストラクタ関数は、既存のPdfファイルを開くために使用されます。新しいPdfファイルを作成する場合でも、テンプレートファイルから作成する必要があります。

## サンプル

```javascript
var pdf = new Pdf("mytemplate.pdf");
```

## API

| 呼び出し | 戻り値 |
|---|---|
| `new Pdf ( path )` | `Pdf` |

| パラメータ | 型 | 説明 |
|---|---|---|
| `path` | `String` | ストレージフォルダからの相対的なPdfファイルパス。 |
