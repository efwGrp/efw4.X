# Batch.concat

The `concat` function is used to concatenate the batch result to another.

## Sample

```javascript
var batch = new Batch();
var batch1 = new Batch();
batch.concat(batch1);
```

## API

| Calling | Returning |
|---|---|
| `Batch . concat ( batch )` | `Batch` |

| Parameters | Type | Description |
|---|---|---|
| `batch` | `Batch` | The sub-batch result. |
