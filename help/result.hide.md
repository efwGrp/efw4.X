# Result.hide

The `hide` function hides elements on the client-side.

## Sample

```javascript
var result = new Result();
result.hide("#table1");
```

## API

| Calling | Returning |
|---|---|
| `result. hide ( selector )` | `Result` |

| Parameters | Type | Description |
|---|---|---|
| `selector` | `String` | A jQuery selector identifying the elements to be hidden. |
