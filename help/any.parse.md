<H1>any.parse</H1>

The parse function is established to parse String to Number or Date.
{any} = { Number | Date }

<h2>Sample</h2>
<pre>
	var nm = Number.parse("12,345.0", "#,000.0");
	var dt = Date.parse("2016/09/08", "yyyy/MM/dd");
</pre>
