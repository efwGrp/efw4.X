<H1>new Record</H1>

The constructor function is established to create a record object.

<h2>Sample</h2>
<pre>
	var record = new Record([
		{"data1":"hello world", "data2":123, "data3":new Date("2016/12/13") },
		{"data1":"hello human", "data2":456, "data3":new Date("2016/12/14") }
	]);
</pre>
